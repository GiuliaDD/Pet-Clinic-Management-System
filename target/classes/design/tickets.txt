PCMS-1
As a user of the system I should be able to add a new veterinarian. I should be able to specify: firstname, lastname, address, specialty.

PCMS-2
As a user of the system I should be able to view all the veterinarians. I should be able to see for each veterinarian: id, firstname, lastname, address, specialty.

PCMS-3
As a user of the system I should be able to delete an existing veterinarian by id.

PCMS-4
As a user of the system I should be able to update an existing veterinarian by id. I should be able to update: firstname, lastname, address, specialty.

---

PCMS-5
As a user of the system I should be able to add a new pet. I should be able to specify: race, birthdate, isVaccinated, ownerName.

PCMS-6
As a user of the system I should be able to view all the pets. I should be able to see for each pet: id, race, birthdate, isVaccinated, ownerName.

PCMS-7
As a user of the system I should be able to delete an existing pet by id.

PCMS-8
As a user of the system I should be able to update an existing pet by id. I should be able to update: race, birthdate, isVaccinated, ownerName.

---

PCMS-9
As a user of the system I should be able to add a new consult. I should be able to specify: veterinarian_id, pet_id, date, description.

PCMS-10
As a user of the system I should be able to view all the consults. I should be able to see for each pet: id, veterinarian_id, pet_id, date, description.

PCMS-11
As a user of the system I should be able to update an existing consult by id.

---
